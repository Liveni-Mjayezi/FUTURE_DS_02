# Customer Churn Analysis Dashboard

## Project Overview
This project was completed as part of a Data Science & Analytics Internship.

The dashboard was developed in Power BI using the Telco Customer Churn dataset to analyze customer behavior and identify factors contributing to customer churn.

## Objectives
- Analyze customer churn rates
- Identify key churn drivers
- Compare churn across customer segments
- Evaluate contract and payment patterns
- Provide actionable business recommendations

## Tools Used
- Power BI
- CSV Dataset
- GitHub

## Files Included
- Task2.pbix – Power BI Dashboard
- WA_Fn-UseC_-Telco-Customer-Churn.csv – Dataset
- task2.pdf – Project Report

## Key Insights
- Churn rate is 26.6%.
- Month to month contracts have the highest churn.
- Electronic check customers churn the most.
- Most churn occurs in the first 12 months.
- Higher monthly charges are linked to higher churn.

## Recommendations
- We should promote longer term contracts.
- Improve onboarding for new customers.
- Encourage automatic payment methods.
- We should offer discounts to high risk customers.
- Reward loyal long term cust.
